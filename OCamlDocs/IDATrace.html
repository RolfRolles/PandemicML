Module IDATrace


module IDATrace: sig .. end
Module for tracing in IDA. Call IDATrace.begin_trace with a tracer to begin tracing. The tracer can be of any data type. The tracer function, "process" (horribly named, process_step maybe?), is called upon every instruction executed. If it returns false, no further steps are taken. Otherwise, the debugger continues executing.

Still haven't decided exactly what this module should look like; should there be an "initialize" function? Maybe objects works better here? (Maybe not; can we interface with them in C?)
