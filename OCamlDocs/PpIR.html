Module PpIR


module PpIR: sig .. end
Pretty printing for terms in the IR.

Pretty-printing functions

val ppTypereg : IR.typereg -> string
Returns the size of a bit-vector quantity.
val ppExpr : bool -> IR.expr -> string
Returns a textual representation of an expression, as well as whether to parenthesize the outermost expression. Since my parser does not have a notion of operator precedence, there are waaaaay too many parentheses in the output for the time being.
val ppInstr : IR.instr -> string
Returns the textual representation of the instruction, along with its subexpressions.
val ppVar : IR.var -> string
Describes a variable. Uses hard-coded variable numbers for x86 registers currently. Think about better ways to do this.
