Module X86ToIR


module X86ToIR: sig .. end
Functions responsible for translating x86 instructions into IR.instr types. There exist two functions. Both take as their first parameter the address at which the instruction is situated. This is used to generate labels within the IR. One function also takes an x86 instruction as input; the other one uses X86Decode.decode to generate the X86.x86instr instructions on the fly.
