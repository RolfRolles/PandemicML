Module X86Util


module X86Util: sig .. end
Miscellaneous stuff relating to X86. I should put everything that is needed to create a "minimal" disassembly string here. I.e., ignore default segments and soforth. For now, it's a bit of a junk pile.

Useful functions

val default_seg : X86.x86_addr_expr -> X86.x86_segreg
Analyze the registers used in a memory expression to determine the default segment that would normally be used by a segment-less instruction encoding. This allows us to not display segment prefixes for instructions that don't require them.
val get_seg : X86.x86_addr_expr -> X86.x86_segreg
Get the segment from a given x86 memory expression.
