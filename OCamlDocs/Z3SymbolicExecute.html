Module Z3SymbolicExecute


module Z3SymbolicExecute: sig .. end
Decision procedure integration: Performs symbolic execution using Z3 over a given list of IR instructions, and given an arbitrary postcondition.
val symbolic_execute : IR.instr list -> IR.expr -> string
Given a list of instructions, turn it into SSA form, asserting each instruction as the assertion of a constraint. So far, this does not handle anything other than assignment statements (i.e. computed gotos and conditional jumps as unsupported). The second argument is a postcondition which is asserted as being true; the variables in the postcondition are updated to match their post-SSA-translation counterparts before being asserted.
